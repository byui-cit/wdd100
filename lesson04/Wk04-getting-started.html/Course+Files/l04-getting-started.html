Step 01

Make sure that you have completed the Software setup activity from week 1. If you have not, do it now. You should already have created your url using updog that is stored in dropbox. Now we are going to begin creating the first page of your website. This will be your home page and will be named index.html.

You will begin by opening "Brackets" (your text editor). In the menubar, choose new. This will open a new page in the editor and you will see that the curser is on line 1. You will need to save this page as index.html. This will become your Home page.

Naming files for the web

There are a few things to keep in mind as you come up with names for the files and folders that will make up your website.

  • Always name the first page of your site index.html
  • Most web hosts are case-sensitive which means there is a difference between Index.html and index.html
  • So...Make your filenames all lowercase to avoid problems with case sensitivity.
  • Avoid spaces and other special characters in your filenames (a-z, 0-9, -, and _ are all safe to use)

Step 02

On line 1, you will type in the first tag:  <!DOCTYPE html> This tells the browser that this will be an html page.

Line 2 will be  <head>. Think of a page of code as having a head, a body, and a foot like a person. A head contains things that will not be seen on the page, but the browser uses them to set things up to show a page.

From the Khan Academy lesson we learned about <meta charset="utf-8">. Meta tags give the browser information to help it build the web page. They are always found in the  <head>. Also found in the with the  <head> yu will find the  <title> which is not displayed on the page. The title shows at the top of the browser window (address bar) or on the tab of a browser window, and in the search listing for a site. Lets pick something that will make sure people know what the site is, and what page they are currently on. Let's make the title something like: Roaring Rapid Vacations || Home (Don't make it too long).

So now, you should have DOCTYPE, html, head, meta charset, and the title tags... each one must have the corresponding closing tag... Now it is time to close the  </head>

Step 04

We have finished the  <head> section so now we will move on to the  <body> (add the body tag next).

Before we start adding items in the body, let's compile the images you plan to use on your page. There are a couple of ways to add images to use in your code. Here are a couple of pages with images that you can use for this assignment:

  • dry oar images 1
  • dry oar images 2 

(or you can find some of your own). Pull that site up and find a logo image you might like to use on your page.

Add an <img> tag inside the <body>. Remember, the <img> tag has 2 required attributes:  src=" " and alt=" ". The alt=" " should contain a very short description about the image (a word or two) and is used primarily for times when the image cannot be displayed for some reason or another.

The logo image in this file is already the right size and is on a web server, so we just need to figure out what it's URL is to add it to our page. Your browser makes it really easy to do that. If you click with the right mouse button on any image in the browser your computer will offer you several actions you can take. The one we are interested in right now is "Copy Image Address".

Copy the address for the logo image and paste the result into the src attribute in our image. the final tag should look something like this:

<img src="http://brothert.net/dryoar/images/dryoarlogo.png" alt="Dry Oar boating logo">

That first method works alright, but what happens if for some reason down the road, the owner of an image removes the image from the server it was on when you made the site, or moves it. It will no longer be able to be called up.

The other method for getting images to use is to save the images to your folder that houses your website in dropbox. This way, they will be there for your use as long as you keep them there with your index.html page. If you choose this method, and I suggest you do, choose one of the images and right click... this time choose save image as: and choose where to save your images (keep the name it already has or choose a new name) ... and save the image... Make sure that if you first save the images to your desktop, that you drag the images to the dropbox folder that contains your index.html file we are creating.

Step 05

Now it's time to add the list of pages that will be in your website... Home and two other pages called subpages or child pagesthat will eventually become links and provide our site's navigation.

Notice that I said 'list' there. One thing most people don't realize is that most navigation bars on most websites are made with an unordered list...the same one you learned how to use in this week's Khan Academy lesson. They are well disguised, but they really are lists.

Take the list of pages from your site plan that you chose (remember, your pages might be different from the 3 pages you see here) and create an unordered list <ul> with each page becoming it's own list item <li> right below the logo that you added.

 <ul> 
 <li> Home </li> 
 <li> Rivers </li> 
 <li> About Us </li> 
 </ul>
