Setup Software Used In Course

Activity Instructions

Step 01  

Review and download the program below: 

  • Google Chrome - You will need to download this software (if you do not already have it installed). It is free.
  • Handcraft - This is the online tool that you will use to create and host your website.  This tool requires the use of the Google Chrome Web Browser.  You will need to make sure you use this browser to build your website.  You may want to save it as a favorite or bookmark it so you can find it more easily.

Starting in Week 4, you will need to submit assignments using Handcraft. To do this, you will need a Gmail account. If you don't already have a Gmail account, please follow these steps to set up a Gmail account.

Later in the semester we will be making wireframes. Here are a couple of tools that can be helpful with that.

  • Wireframe.cc Lightweight online tool for creating wireframes.
  • Cacoo.com Another online wireframing tool with a larger feature set.
