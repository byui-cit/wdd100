Step 02

Learn the vocabulary.

HTML has some very specific vocabulary that comes with it. It is important that we become familiar with it. Spend a few minutes familiarizing yourself with the following list. You will have a hard time making sense of the instruction given in this course if you don't understand these terms.

Tag
HTML tags are key words surrounded by angle brackets: ie <body>
Element
An HTML element usually consists of a start tag and end tag, with the content inserted in between
Nesting
Elements can be placed inside of other elements.
Parent
In nesting the parent is the element that contains other elements.
Child
In nesting a child element is contained inside of another element
Attribute
Attributes provide additional information about HTML elements.
Semantics
Semantics is the study of the meanings of words and phrases in a language. Semantic elements = elements with a meaning.

Watch for mention of the terms above as you review the following short tutorials.

  • Introduction to HTML
  • HTML Basics
  • HTML Elements
  • HTML Attributes
  • HTML Semantics

Step 03

Let's write some HTML. The Introduction to HTML link above contained a nice starting point. Below you can find the same thing. Type the HTML below into the new file we created in your editor.


<!doctype html>
<html>
    <head>
          <title>All About Me</title>
    </head>
    <body>
        <p>Hello World!</p>
    </body>
</html>

This code contains everything you need for a basic web page. It has most of the items listed above. Tags, elements, nesting,parent elements, child elements, and semantics. Can you spot them all?

Step 04

A visitor to a website should not have to guess what the page is about. To let them know what our page is, let's add a headline advertising it. We can use the <h1> tag for this. Add a line between the <body> and <p> elements. Add an <h1> opening tag, then type "About Me", then enter </h1> to close the element.

Your code should look like this:


<!doctype html>
<html>
    <head>
          <title>All About Me</title>
    </head>
    <body>
        <h1>About Me</h1>
        <p>Hello World!</p>
    </body>
</html>
