Setup Software Used In Course

Activity Instructions

One of our goals for this class is to allow you to create a real website accessible online from the internet. This, as you will discover in the readings for this week, requires that you have what is know as a host. Most hosts cost money, but not all. For this class we will be using a free service called HTMLDrop as our host. You can then use one of several good text editors to write your actual HTML with. I recommend one called Brackets if you don't have another that you like.

There are a few steps we will need to do to get everything setup.

Step 01  

HTMLDrop uses Dropbox for the file storage. If you already have a Dropbox account, and their desktop program setup on your computer and know how to use it, skip to the next step. If you do not then follow the instructions on setting up Dropbox, then come back.

Step 02

Goto HTMLDrop.com and sign in with your Dropbox credentials. HTMLDrop will need to create a folder inside your Dropbox folder called "HTMLDrop" and will ask for permission to do this. After granting it permission it will take you to an account screen.

Step 03

Next we need to create a domain for you to use. Click on the "Create" button on the right side of the screen. Enter whatever you want to use for your domain. This does need to be unique, and will only allow letter and numbers. I usually recommend something like "semesterLastnameFirstinitial". So for me it would be "w16thompsond" (if it were Winter 2016).

This will form the URL you will turn in for most of your assignments over the course. You can always find it on the HTMLDrop Account screen if you forget it.

When you created your domain in HTMLDrop it automatically created a folder on your computer where you need to place all of your files for you webpages. In my case the folder is: DropBox/Apps/HTMLDrop/w16thompsond. If you place your files anywhere else your instructor will not be able to see them to give you a grade!

Step 04  

Download and install Brackets, then complete the Brackets setup instructions.

Here is the complete list of software we will use this semester:

  • Dropbox: Cloud based file storage service. Great way to backup important files and make them available on any computer.
  • HTMLDrop: Service that works with Dropbox to make it work as a web host.
  • Brackets (Mac and Windows) Text Editor made specifically for working with HTML and CSS.
  • Wireframe.cc Lightweight online tool for creating wireframes.
  • Cacoo.com Another online wireframing tool with a larger feature set.
