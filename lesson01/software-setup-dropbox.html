Setup Software Used In Course

Activity Instructions

One of our goals for this class is to allow you to create a real website accessible online from the internet. This, as you will discover in the readings for this week, requires that you have what is know as a host. Most hosts cost money, but not all. For this class we will be using a free service called HTMLDrop as our host. You can then use one of several good text editors to write your actual HTML with. I recommend one called Brackets if you don't have another that you like.

There are a few steps we will need to do to get everything setup.

Step 01  

HTMLDrop uses Dropbox for the file storage. If you already have a Dropbox account, and their program setup on your computer and know how to use it, skip to the next step. If you do not then follow the instructions on setting up Dropbox, then come back.

Step 02

Goto HTMLDrop.com and sign in with your Dropbox credentials. HTMLDrop will need to create a folder inside your Dropbox folder called "HTMLDrop" and will ask for permission to do this. After granting it permission it will take you to an account screen.

Step 03

Next we need to create a domain for you to use. Click on the "Create" button on the right side of the screen. Enter whatever you want to use for your domain. This does need to be unique, and will only allow letter and numbers. I usually recommend something like "semesterLastnameFirstinitial". So for me it would be "w16thompsond" (if it were Winter 2016).
