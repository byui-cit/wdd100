Setup Software Used In Course

Activity Instructions

One of our goals for this class is to allow you to create a real website accessible online from the internet. This, as you will discover in the readings for this week, requires that you have what is know as a host. Most hosts cost money, but not all. For this class we will be using a service called github pages as our host. You can then use one of several good text editors to write your actual HTML with. I recommend one called Brackets if you don't have another that you like.

There are a few steps we will need to do to get everything setup.

Step 01  

Watch and follow the instructions in the following youtube videos by Brother Robertson. There are 3 videos that range from 3-6 minutes long that will walk you through all of the setup for github.

Starting in Week 4, you will need to submit assignments using Github. Make sure that you watch the videos and do the setup before then.

With the link below download and install Brackets. There are also links to a couple of services we will use later in the semester for making wireframes.

  • Brackets (Mac and Windows) Text Editor made specifically for working with HTML and CSS.
  • Wireframe.cc Lightweight online tool for creating wireframes.
  • Cacoo.com Another online wireframing tool with a larger feature set.
