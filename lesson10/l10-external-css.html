External CSS

Activity Instructions

Estimated Time: 15 minutes

Up until now we have been using an internal stylesheet. This is what Khan Academy teaches, and it is a good place to start learning. Eventually you can start running into some limitations of internal CSS...namely that when you want to add more than one page to your site keeping your styling in sync between all your pages becomes quite a chore as every page will have a copy of the styles in the <head>!

External style sheets solve this problem by removing the CSS from the page and into a separate file of type .css After removing the css you then have to <link> your html to the css so it can find the styles.

Step 01

Make your CSS external by first creating a new file in your editor and saving it as main.css.

Step 02

Next copy your CSS out of your index.html file (it will be everything in the <style> block, but not the <style> tags themselves) and pasting the CSS into your new main.css file.

Step 03

Now link your css to your html page by adding a <link> tag. First remove the <style> block from the <head> of your document. The link will go in the place of it. A link tag has 3 required attributes: rel, type, and href. Two of the attributes will always be the same for CSS, the third you will have to provide. In this case the line should look like this (assuming you named your file main.css):

<link rel="stylesheet" type="text/css" href="main.css" >

Step 04

Enter the URL to your site into a browser to make sure the webpage displays correctly. Once verified, go to the next page to access "W10.01 External CSS" and click "Upload" to submit the URL for this assignment.

If you have not done so recently make sure to backup your site! (In Handcraft goto Projects, then click on the gear icon on your project for the class, then "Export as .zip"

Grading

This activity will be graded using the following rubric:

  • All CSS has been removed from the HTML file and placed into an external CSS file (7 pts)
  • CSS is linked to the html file and styles still show. (3 pt)
