Setup Brackets

Activity Instructions

Brackets is an editor designed specifically for writing HTML and CSS. Setting it up is fairly simple...but there are a few steps you should do after installation

Step 01

If you haven’t already, download and install Brackets. http://brackets.io

Step 02

When you first launch Brackets it will have an example site open. You should see something like this:

Notice the "Getting Started" text. This is the currently open project. You need to change this to your updog Folder. Click on the Getting Started, then "Open Folder". In the Search box at the top of the Open Dialog type "updog". You should see your updog folder show up...click on it to open it. Now you should see a folder with the name that you cose for your URL during the software setup activity. If you don't see a folder go back and do that again. Select the folder then click "open".

Make sure when you are working that you always see your folder name in Brackets. As long as you have your project open all your files will save in the right place. If you place your files anywhere else (Path should be something like DropBox/Apps/updog/w17thompsond) your instructor will not be able to see them to give you a grade!

Step 03

Create a new file by clicking File->New or pressing cmd-N on the mac. Ctrl-N on the PC. Save your file as "index.html". All lowercase!

Step 04

You can now start entering your html. Could look something like below:


<!-- HTML - File Name: "index.html" -->
<!doctype html>
<html>
    <head>
          <title>My first website</title>
    </head>
    <body>
        <p>Hello World!</p>
    </body>
</html>

Step 05

If you have Chrome installed on your computer you can click on the Live Preview button on the right side of the edit window and it will bring your page up in that browser for you to see. As long as you don’t have any errors in your html it should update real-time as you make changes. If you don’t have Chrome installed, open the folder that contains the html file and right click->Open with and select your preferred browser.

Step 06

Open a new window/tab in your browser and enter the HTMLDrop URL in. For me this would be https://w17thompsond.updog.co. You should see the same thing there that you did when you previewed in Brackets. That is the URL you will turn in for your assignments.
